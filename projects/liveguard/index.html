Project 02 / Real-time Service

LiveGuard

面向实时内容场景的智能监测与风险提示原型，尝试将事件处理、规则判断和实时反馈组合成清晰流程。

  • Java
  • Spring Boot
  • Redis
  • WebSocket

Project Overview

LiveGuard 是一个轻量的实时监测系统原型，重点练习事件接入、状态缓存、风险判断与前端通知之间的协作。

Core Features

  • 实时事件接入与状态更新。
  • 可配置的基础风险判断流程。
  • 通过长连接向页面推送提示。

Highlights

  • 清晰区分数据接入、判断与通知职责。
  • 使用缓存支持短周期状态管理。
  • 保持原型简洁，方便后续替换具体能力。

Future Plans

补充事件回放、基础指标与规则管理界面，并完善异常状态下的降级处理。
