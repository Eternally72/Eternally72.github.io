Project 04 / Backend System

Microservice Trading Platform

用于学习服务拆分、数据一致性与可观测性的交易平台实践，重点理解分布式后端系统的基础边界。

  • Java
  • Spring Cloud
  • MySQL
  • Redis
  • Docker

Project Overview

这是一个面向后端工程学习的微服务项目，通过有限的业务模块练习服务治理、缓存、持久化和容器化运行。

Core Features

  • 用户、订单与资产等基础服务拆分。
  • 缓存与数据库协同的常见数据流程。
  • 统一配置、服务调用与容器化运行。

Highlights

  • 从单体思维过渡到明确的服务边界。
  • 关注幂等、异常处理与可观察性。
  • 项目结构适合作为持续演进的练习环境。

Future Plans

补充链路追踪、基础压测与更完整的故障处理记录，逐步验证系统在不同条件下的表现。
