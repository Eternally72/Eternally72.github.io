Project 01 / Multi-Agent System

AIWerewolf

基于 Java 21、Spring Boot 和 Vue 3 构建的多 Agent 狼人杀系统，支持真人参与、真人观战、真人与 AI 混合对局，以及全 AI 自动对局。

  • Java 21
  • Spring Boot 3
  • Vue 3
  • TypeScript
  • MySQL
  • Redis
  • WebSocket/STOMP
  • Maven

Project Overview

项目以后端状态机驱动完整对局流程，为每个 AI 玩家维护独立角色、视角、记忆和策略，并通过浏览器端游戏桌呈现发言、投票、夜间行动与观战过程。

Core Features

  • 支持真人参与、真人观战、真人与 AI 混合对局及全 AI 自动模拟。
  • 通过 GameViewBuilder 隔离角色信息，避免普通玩家或 Agent 获取完整真相。
  • 使用 REST 恢复状态，并通过 WebSocket/STOMP 推送阶段、时间线和公共事件。
  • 多模型网关支持 Mock 与多种兼容 Provider，默认无需真实 API Key 即可运行。

Highlights

  • GamePhaseEngine 与行动服务拆分回合推进、昼夜行动、投票和死亡结算。
  • MySQL 保存长期事实与审计记录，Redis 负责运行态、锁、幂等键和短期记忆。
  • GodView 可查看身份、AgentRun、AgentTask 与公开或完整视角的事件回放。
  • 真实模型密钥仅通过后端环境变量读取，前端不会直接接触敏感配置。
