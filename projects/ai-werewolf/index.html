Project 01 / Multi-Agent System

AIWerewolf

围绕狼人杀场景构建的多智能体协作、推理与交互实验，用可观察的游戏流程探索 Agent 行为。

  • Python
  • FastAPI
  • Multi-Agent
  • WebSocket

Project Overview

项目将角色决策、回合推进和实时交互拆分为清晰模块，重点关注多智能体在有限信息环境下的协作与推理表现。

Core Features

  • 多角色 Agent 的独立上下文与行动流程。
  • 可观察的游戏状态、消息和回合管理。
  • 面向浏览器交互的实时通信接口。

Highlights

  • 用具体场景验证多 Agent 编排思路。
  • 服务端结构便于替换模型与策略。
  • 保持演示体验与工程可维护性的平衡。

Future Plans

继续完善行为评测、对局回放与策略对比，让实验结果更容易观察和复现。
