Project 03 / Campus Service

Campus Multi-Agent System

面向校园信息与服务场景的多智能体系统，让不同能力的 Agent 协作完成检索、规划与任务处理。

  • Python
  • LLM
  • RAG
  • Agent
  • API

Project Overview

项目围绕校园常见信息需求进行能力拆分，通过统一编排层协调检索、问答与任务型 Agent，探索更自然的服务入口。

Core Features

  • 根据用户意图选择合适的处理 Agent。
  • 结合本地知识内容完成基础检索与回答。
  • 统一记录任务过程和最终结果。

Highlights

  • 以清晰的校园场景约束系统边界。
  • 不同 Agent 能力可以独立替换与扩展。
  • 关注回答来源、流程可观察性和可维护性。
